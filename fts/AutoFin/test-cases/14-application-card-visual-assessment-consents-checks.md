2. В появившемся поле `Комментарий` ввести значение из тестовых данных.
3. Снять checkbox `Другое`.
4. Повторно отметить checkbox `Другое` в том же блоке без закрытия формы.

### Итоговый ожидаемый результат

### Постусловия

## TC-ACVCC-018

**Название:** Сообщение обязательности для пустого комментария при выбранном `Другое`

**Тип:** Negative

**Приоритет:** High

**package_id:** WP-01

**Трассировка:** `ATOM-019`; `SRC-003`; `BSR 317`; `DICT-001`

### Предусловия

- Открыта карточка заявки.
- В поле `Визуальная информация` выбрано значение `Да`.
- Список `Параметры визуальной оценки` отображается.
- В одном из блоков 1-7 списка отображается checkbox `Другое (комментарий обязателен)`.

### Тестовые данные

- Блок списка с checkbox `Другое (комментарий обязателен)`: любой отображенный блок 1-7 из Приложения 1.
- Значение поля `Комментарий`: пустое.

### Шаги

1. Отметить checkbox `Другое (комментарий обязателен)` в выбранном блоке.
2. Перевести фокус в появившееся поле `Комментарий`.
3. Не заполняя поле `Комментарий`, перевести фокус из поля в другой элемент формы.

### Итоговый ожидаемый результат

После потери фокуса пустым полем `Комментарий` отображается сообщение `Обязательно к заполнению`.

### Постусловия

- Если состояние карточки сохраняется, снять checkbox `Другое` и вернуть поле `Визуальная информация` в исходное значение `Нет`.
