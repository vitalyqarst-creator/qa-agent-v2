## TC-AF-DOC-001
**Название:** Отображение блока документов и информационного поля анкеты клиента
**Тип:** Positive
**Приоритет:** High
**package_id:** WP-01
**Трассировка:** ATOM-001; ATOM-002; ATOM-003; SRC-001; SRC-002; BSR 206; BSR 207

### Предусловия
Открыта карточка заявки в разделе «Заявка».

### Тестовые данные
Не требуются.

### Шаги
1. Перейти к блоку «Документы по заявке».
2. Найти информационное поле «Анкета клиента. Распечатайте, подпишите с клиентом и загрузите скан в заявку».

### Постусловия
Не требуются.

## TC-AF-DOC-002
**Название:** Загрузка анкеты клиента через проводник в поддерживаемых форматах
**Тип:** Positive
**Приоритет:** High
**package_id:** WP-01
**Трассировка:** ATOM-004; ATOM-005; ATOM-008; ATOM-011; SRC-003; BSR 208; BSR 209; BSR 210; BSR 211

### Предусловия
Открыт блок «Документы по заявке» без ранее прикрепленного файла в поле «Анкета клиента».

3. В поле «Анкета клиента» добавить файл `client-questionnaire.png` через drag and drop.
4. Удалить добавленный файл через доступный UI-механизм удаления вложения.
5. В поле «Анкета клиента» добавить файл `client-questionnaire.pdf` через кнопку открытия проводника.

### Итоговый ожидаемый результат

### Постусловия

## TC-AF-DOC-028
**Название:** Обязательность второго документа зависит от типа программы
**Тип:** Business
**Приоритет:** High
**package_id:** WP-01
**Трассировка:** ATOM-028; SRC-006; BSR 225

### Предусловия
Открыт блок «Документы по заявке»; поле «Тип документа» доступно; известен тип программы заявки.

### Тестовые данные
Проверяемые варианты:
- программа не ГОС;
- программа ГОС;
- программа для инвалидов.

### Шаги
1. Для заявки с программой не ГОС оставить поле «Второй документ» без вложения и проверить обязательность.
2. Для заявки с программой ГОС выбрать `ВУ` как тип второго документа и проверить обязательность загрузки.
3. Для заявки с программой для инвалидов проверить допустимость типов второго документа `ВУ` и `СНИЛС`.

### Итоговый ожидаемый результат
Для не ГОС программы второй документ не обязателен. Для ГОС программы обязательным вторым документом является `ВУ`. Для программы для инвалидов обязательным вторым документом является `ВУ` или `СНИЛС`.

### Постусловия
Очистить выбранные значения и вложения второго документа при необходимости.
