# Canary: поля ФИО контактного лица

## Scope

- FT package: `AutoFin`.
- Main source: `fts/AutoFin/source/FT4AutoFinFinal.xhtml`.
- Structural cross-check: `fts/AutoFin/source/FT4AutoFinFinal.pdf`.
- Scope slug: `application-card-client-addresses-contacts`.
- Canary scope: input restrictions for contact-person fields `Фамилия`, `Имя`, `Отчество`.
- Source rows:
  - `SRC-CANARY-174`: field `Фамилия`, BSR 174: "Возможен ввод только текстовых символов и специальный символ «-»."
  - `SRC-CANARY-176`: field `Имя`, BSR 176: "Возможен ввод только текстовых символов и специальный символ «-»."
  - `SRC-CANARY-178`: field `Отчество`, BSR 178: "Возможен ввод только текстовых символов и специальный символ «-»."

## Setup Profiles

### SETUP-CANARY-AUTOFIN-EDITABLE-DRAFT

1. Создать или взять из тестового стенда редактируемую черновую заявку AutoFin с валидными обязательными данными, достаточными для открытия карточки заявки.
2. Открыть карточку заявки в пользовательском интерфейсе.
3. Перейти в блок анкеты, где доступен виджет `Добавить контактное лицо`.
4. Не сохранять изменения автоматически после ввода значений в поля контактного лица.
