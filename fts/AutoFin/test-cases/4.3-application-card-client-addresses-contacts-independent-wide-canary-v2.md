# Тест-кейсы: карточка заявки, адреса и контакты клиента

Набор покрывает видимые UI-обязанности раздела 4.3 по адресам клиента, контактам клиента и контактным лицам. Диагностические материалы v2 находятся в `fts/AutoFin/work/canary-runs/independent-wide-canary-v2/`.

## TC-AF43-AW2-001

**Название:** Поле "Адрес регистрации" показывает найденный адрес через DaData
**Тип:** Positive
**Приоритет:** High
**package_id:** WP-01
**Трассировка:** BSR 115; BSR 116; BSR 119; Table 7 row 33; ATOM-AW2-001

**Предусловия:**
1. Открыть карточку заявки, доступную для редактирования.
2. Перейти к блоку "Адреса клиента".
3. Убедиться, что признак ручного ввода адреса регистрации установлен в значение "Нет".

**Тестовые данные:**
- Адрес регистрации: `г Красноярск, ул Мира, д 10`.

**Шаги:**
1. В поле "Адрес регистрации" ввести `г Красноярск, ул Мира, д 10`.
2. Выбрать найденный адрес из подсказок.

**Итоговый ожидаемый результат:**
В поле "Адрес регистрации" отображается выбранный адрес `г Красноярск, ул Мира, д 10`.

**Постусловия:**
Закрыть карточку заявки без сохранения изменений.
