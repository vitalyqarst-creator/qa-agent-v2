## TC-002

**Название:** Обязательность поля «Серия» текущего паспорта при пустом значении
**Тип:** негативный
**Приоритет:** высокий
**package_id:** WP-01
**Трассировка:** OBL-002; ATOM-020; SRC-020
**Статус oracle:** ui-calibration-required
**Статус тест-кейса:** candidate-ui-calibration
**Требуется подтверждение:** Какое действие запускает проверку обязательности поля «Серия» и какая наблюдаемая UI-реакция подтверждает, что пустое значение не принято как валидное для продолжения?

### Предусловия

1. Открыть анкету клиента.
2. Перейти к блоку «Паспортные данные», содержащему доступное поле «Серия» текущего паспорта.

### Тестовые данные

- Пустое значение поля «Серия».

### Шаги

1. Оставить поле «Серия» пустым.

### Итоговый ожидаемый результат

Пустое обязательное поле «Серия» не принимается как валидное для продолжения. Во время UI-калибровки необходимо зафиксировать точное проверочное действие, видимое состояние поля и наблюдаемый исход; конкретный механизм реакции требованиями не определён.

### Постусловия

- Не применимо: сохранение или переход требованиями для этой проверки не определены.
