# Тест-кейсы

## TC-AUTOFIN-CALCULATOR-001

**Название:** Открытие окна «Кредитный калькулятор» с незаполненными данными
**Тип:** позитивный
**Приоритет:** средний
**package_id:** WP-01
**Трассировка:** OBL-001; ATOM-001; SRC-006; BSR 39

### Предусловия

1. Открыть раздел «Заявки в системе», содержащий действие «Кредитный калькулятор».

### Тестовые данные

- Не требуются: проверяется исходное незаполненное состояние данных.

### Шаги

1. Нажать кнопку «Кредитный калькулятор».

### Итоговый ожидаемый результат

Открывается окно «Кредитный калькулятор» с незаполненными данными.

### Постусловия

- Не применимо: требование не определяет изменение состояния системы.
