## TC-AFDRP-008

**Название:** Распознавание документа при наличии файла запускает запрос и заполняет поля заявки

**Тип:** Positive

**Приоритет:** High

**package_id:** WP-01

**Трассировка:** `ATOM-008`; `BSR 83`; `SRC-001`; `GAP-002`; `WP-01`

### Предусловия

- Открыто всплывающее окно распознавания документов.
- Система распознавания документов доступна на стенде или заменена управляемым stub-сервисом.
- Есть способ проверить факт запроса в систему распознавания: лог, stub history, API evidence или иной согласованный артефакт.

### Тестовые данные

- Файл документа поддерживаемого формата: `recognition-passport.pdf`.
- Fixture распознавания возвращает значения для полей заявки, соответствующих выбранному типу документа.

### Шаги

1. Добавить файл `recognition-passport.pdf` в контейнер файлов popup.
2. Выбрать соответствующий тип документа.
3. Нажать кнопку `Распознать`.
4. Проверить evidence запроса в систему распознавания.
5. Проверить поля заявки, для которых fixture вернул значения.

### Итоговый ожидаемый результат

При наличии файла система выполняет запрос в систему распознавания. После успешного ответа распознанные значения из fixture заполнены в соответствующих полях заявки.

### Постусловия

Очистить заполненные распознаванием поля или отменить несохраненные изменения.
