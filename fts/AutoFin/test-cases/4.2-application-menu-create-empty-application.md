# Тест-кейсы

## TC-AUTOFIN-APPLICATIONS-001

**Название:** Открытие карточки заявки с пустыми данными по кнопке «Создать заявку»
**Тип:** позитивный
**Приоритет:** средний
**package_id:** WP-01
**Трассировка:** OBL-001; ATOM-001; SRC-006; BSR 38

### Предусловия

1. Открыть раздел «Заявки в системе».

### Тестовые данные

- Не требуются: ввод данных при создании пустой заявки не выполняется.

### Шаги

1. Нажать видимую кнопку «Создать заявку».

### Итоговый ожидаемый результат

Открывается карточка заявки с пустыми данными.

### Постусловия

- Не применимо: требование определяет только открытие карточки заявки с пустыми данными.
