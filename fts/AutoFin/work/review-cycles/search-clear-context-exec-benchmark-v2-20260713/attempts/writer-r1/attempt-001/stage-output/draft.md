# Тест-кейсы

## TC-SCCB-001

**Название:** Очистка изменённого состояния фильтров поиска
**Тип:** позитивный
**Приоритет:** средний
**package_id:** search-clear-context-exec-benchmark-v2
**Трассировка:** OBL-001; ATOM-001; BSR 32; SRC-001; SRC-001.P01

### Предусловия

1. Открыта область поиска с фильтрами; зафиксировано её исходное состояние фильтров.
2. Доступно хотя бы одно видимое значение фильтра, изменяющее состояние фильтров (`FIX-SCCB-001`).

### Тестовые данные

- Одно доступное видимое значение фильтра, изменяющее состояние фильтров.

### Шаги

1. Выбрать доступное видимое значение фильтра.
2. Убедиться, что состояние фильтров изменилось относительно зафиксированного исходного состояния.
3. Нажать `Очистить`.

### Итоговый ожидаемый результат

После нажатия `Очистить` изменённое состояние фильтров совпадает с зафиксированным исходным состоянием.

### Постусловия

- Не требуются: состояние фильтров восстановлено действием `Очистить`.
