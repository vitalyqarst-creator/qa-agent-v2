## TC-PDCR-001

**Название:** Поле «Фамилия» принимает текстовое значение «Иванов»
**Тип:** позитивный
**Приоритет:** средний
**package_id:** personal-data-character-restrictions-v4
**Трассировка:** OBL-001; ATOM-001

### Предусловия

1. Открыта форма с доступным для ввода полем «Фамилия».

### Тестовые данные

- Фамилия: `Иванов`.

### Шаги

1. Ввести `Иванов` в поле «Фамилия».

### Итоговый ожидаемый результат

Поле «Фамилия» принимает текстовое значение `Иванов`.

### Постусловия

- Не требуются.
