## TC-PREP-001

**Название:** Ограничение одиночного выбора в виджете типа «Список»  
**Тип:** позитивный  
**Приоритет:** средний  
**package_id:** `autofin-widget-selection-prepared-v2`  
**runtime_fixture:** `single-select-widget-with-two-values`  
**Трассировка:** `ATOM-PREP-001`, `SRC-001`

### Предусловия

1. Runtime fixture указывает доступный для проверки UI-контекст с виджетом типа «Список».
2. Виджет настроен на справочник, в котором доступны как минимум два разных значения.
3. UI-контекст открыт, виджет доступен для выбора значений.

### Тестовые данные

- Параметр `V1`: любое доступное в виджете значение настроенного справочника.
- Параметр `V2`: другое доступное в том же виджете значение настроенного справочника; `V2 != V1`.

### Шаги

1. Открыть список доступных значений виджета.
2. Выбрать значение `V1`.
3. Повторно открыть список доступных значений виджета.
4. Выбрать значение `V2`.

### Итоговый ожидаемый результат

После последовательного выбора двух разных доступных значений виджет содержит ровно одно выбранное значение.

### Постусловия

- Не применяются: проверка ограничена текущим состоянием виджета; поведение сохранения и внешних систем не входит в scope.

## TC-PREP-003

**Название:** Начальное пустое состояние нового виджета  
**Тип:** позитивный  
**Приоритет:** средний  
**package_id:** `autofin-widget-selection-prepared-v2`  
**runtime_fixture:** `fresh-uninitialized-widget`  
**Трассировка:** `ATOM-PREP-003`, `SRC-003`

### Предусловия

1. Runtime fixture указывает доступный для проверки UI-контекст со вновь инициализированным подходящим виджетом.
2. После инициализации с виджетом не выполнялись действия выбора или очистки значения.
3. UI-контекст открыт без взаимодействия с проверяемым виджетом.

### Тестовые данные

- Параметр `W`: вновь инициализированный виджет, предоставленный runtime fixture для проверки начального состояния.

### Шаги

1. Не взаимодействуя с виджетом `W`, проверить область отображения его выбранного значения.

### Итоговый ожидаемый результат

До первого взаимодействия с новым виджетом в нём отсутствует выбранное значение.

### Постусловия

- Не применяются: проверка не изменяет состояние виджета и не охватывает внутреннее представление пустого значения.
