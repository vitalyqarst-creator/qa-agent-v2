# Тест-кейсы

## TC-VACS-001

**Название:** Начальное значение поля «Визуальная информация» — «Нет»
**Тип:** позитивный
**Приоритет:** средний
**package_id:** visual-assessment-conditional-v1
**Трассировка:** OBL-COND-001; ATOM-COND-001

### Предусловия

1. Открыта форма с блоком, содержащим поле «Визуальная информация».

### Тестовые данные

- Не требуются.

### Шаги

1. Открыть блок с полем «Визуальная информация» без изменения его значения.

### Итоговый ожидаемый результат

При первом открытии блока поле «Визуальная информация» отображает значение «Нет».

### Постусловия

- Не требуются.

## TC-VACS-004

**Название:** Отображение комментария при выборе «Другое» в категории «Признаки алкоголика»
**Тип:** позитивный
**Приоритет:** средний
**package_id:** visual-assessment-conditional-v1
**Трассировка:** OBL-COND-004; ATOM-COND-004

### Предусловия

1. Открыта форма с полем «Визуальная информация».
2. В поле «Визуальная информация» выбрано значение «Да», список «Параметры визуальной оценки» отображается.

### Тестовые данные

- Категория: «Признаки алкоголика».
- Значение: «Другое (комментарий обязателен)».

### Шаги

1. В категории «Признаки алкоголика» выбрать значение «Другое (комментарий обязателен)».

### Итоговый ожидаемый результат

В категории «Признаки алкоголика» отображается связанное текстовое поле комментария.

### Постусловия

- Не требуются.

## TC-VACS-005

**Название:** Одновременный выбор двух обычных значений категории «Признаки алкоголика»
**Тип:** позитивный
**Приоритет:** средний
**package_id:** visual-assessment-conditional-v1
**Трассировка:** OBL-COND-005; ATOM-COND-005

### Предусловия

1. Открыта форма с полем «Визуальная информация».
2. В поле «Визуальная информация» выбрано значение «Да», список «Параметры визуальной оценки» отображается.

### Тестовые данные

- Категория: «Признаки алкоголика».
- Значение 1: «Запах алкоголя / перегара / сильный запах духов, перебивающий перегар».
- Значение 2: «Отечность, нездоровый цвет лица, синяки под глазами».

### Шаги

1. В категории «Признаки алкоголика» выбрать значение «Запах алкоголя / перегара / сильный запах духов, перебивающий перегар».
2. В этой же категории выбрать значение «Отечность, нездоровый цвет лица, синяки под глазами».

### Итоговый ожидаемый результат

Оба выбранных значения категории «Признаки алкоголика» одновременно отмечены.

### Постусловия

- Не требуются.
