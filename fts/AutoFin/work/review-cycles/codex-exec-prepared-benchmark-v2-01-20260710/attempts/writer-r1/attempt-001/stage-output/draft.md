Статус набора: `draft`, unsigned.

## TC-PREP-001

**Название:** Ограничение одиночного выбора в виджете типа «Список»  
**Тип:** позитивный  
**Приоритет:** средний  
**Статус:** draft, unsigned  
**package_id:** autofin-widget-selection-prepared-v2  
**execution_profile:** simple-field-property  
**Трассировка:** ATOM-PREP-001  
**Источник:** SRC-001

### Предусловия

1. Подготовлен доступный пользователю runtime fixture с новым виджетом типа «Список», соответствующим выбранному scope.
2. В виджете доступны как минимум два разных значения из подключенного к нему справочника.
3. Значения в виджете ранее не выбирались.

### Тестовые данные

- `Значение A` — первое доступное значение из фактически отображаемого в виджете справочника.
- `Значение B` — другое доступное значение из того же справочника, не равное `Значению A`.

### Шаги

1. Открыть runtime fixture с виджетом типа «Список».
2. Открыть перечень доступных значений виджета.
3. Выбрать `Значение A`.
4. Повторно открыть перечень доступных значений виджета.
5. Выбрать `Значение B`.

### Итоговый ожидаемый результат

После последовательного выбора двух разных доступных значений виджет содержит ровно одно выбранное значение.

### Постусловия

- Не применимо: кейс не включает сохранение или изменение данных за пределами runtime fixture.

## TC-PREP-003

**Название:** Начальное пустое состояние нового виджета  
**Тип:** позитивный  
**Приоритет:** средний  
**Статус:** draft, unsigned  
**package_id:** autofin-widget-selection-prepared-v2  
**execution_profile:** simple-field-property  
**Трассировка:** ATOM-PREP-003  
**Источник:** SRC-003

### Предусловия

1. Подготовлен доступный пользователю runtime fixture, создающий новый виджет одного из типов выбранного scope.
2. Для нового виджета не задано начальное значение.
3. Пользователь еще не взаимодействовал с новым виджетом.

### Тестовые данные

- `Целевой виджет` — новый виджет типа «Список» или «Список с множественным выбором» из runtime fixture.

### Шаги

1. Открыть runtime fixture с новым целевым виджетом, не выполняя действий выбора в этом виджете.
2. Проверить отображаемое значение целевого виджета.

### Итоговый ожидаемый результат

До первого взаимодействия с новым виджетом в нем отсутствует выбранное значение.

### Постусловия

- Не применимо: состояние виджета не изменялось.
