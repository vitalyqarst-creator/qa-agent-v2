# Тест-кейсы

## TC-PDSP-001

**Название:** Проверка постоянной видимости поля «Фамилия»
**Тип:** позитивный
**Приоритет:** средний
**package_id:** personal-data-static-properties-v3
**Трассировка:** OBL-001; ATOM-001

### Предусловия

1. Пользователь имеет доступ к карточке заявки.

### Тестовые данные

- Не требуются.

### Шаги

1. Открыть карточку заявки.

### Итоговый ожидаемый результат

Поле `Фамилия` отображается.

### Постусловия

- Не применимо: проверка не изменяет данные.
