## TC-PREP-001

**Название:** Ограничение одиночного выбора в виджете типа «Список»  
**Тип:** позитивный  
**Приоритет:** средний  
**package_id:** `autofin-widget-selection-prepared-v3`  
**Трассировка:** `ATOM-PREP-001` (`SRC-001`)

### Предусловия

1. Открыт предоставленный runtime fixture с новым виджетом типа «Список»; до выполнения кейса значения в виджете не выбирались.
2. В runtime fixture виджет доступен для взаимодействия и содержит не менее двух различных доступных значений.

### Тестовые данные

- `Значение A` — одно доступное значение из runtime fixture.
- `Значение B` — другое доступное значение из того же runtime fixture, отличное от `Значения A`.

### Шаги

1. В виджете типа «Список» выбрать `Значение A`.
2. В том же виджете выбрать `Значение B`.
3. Проверить количество значений, которые виджет отображает как выбранные.

### Итоговый ожидаемый результат

После последовательного выбора двух разных доступных значений виджет содержит ровно одно выбранное значение.

### Постусловия

- Специальное восстановление данных не требуется: проверка ограничена runtime fixture; сохранение и persistence не выполняются и не проверяются.

1. Открыт предоставленный runtime fixture с новым виджетом типа «Список с множественным выбором»; до выполнения кейса значения в виджете не выбирались.
2. В runtime fixture виджет доступен для взаимодействия и содержит не менее двух различных доступных значений.

### Тестовые данные

- `Значение A` — одно доступное значение из runtime fixture.

## TC-PREP-003

**Название:** Начальное пустое состояние нового виджета  
**Тип:** позитивный  
**Приоритет:** средний  
**package_id:** `autofin-widget-selection-prepared-v3`  
**Трассировка:** `ATOM-PREP-003` (`SRC-003`)

### Предусловия

1. Открыт предоставленный runtime fixture с новым виджетом, к которому применимо правило начального состояния из `ATOM-PREP-003`.
2. После создания или инициализации runtime fixture с проверяемым виджетом ещё не взаимодействовали.

### Тестовые данные

- `Проверяемый виджет` — новый виджет из runtime fixture, к которому применимо `ATOM-PREP-003`.

### Шаги

1. Не выполняя выбора или ввода значения, проверить текущее содержимое `Проверяемого виджета`.

### Итоговый ожидаемый результат

До первого взаимодействия с новым виджетом в нём отсутствует выбранное значение.

### Постусловия

- Не применимо: тест не изменяет состояние виджета; внутренняя интерпретация пустого значения как `NULL` не проверяется.
