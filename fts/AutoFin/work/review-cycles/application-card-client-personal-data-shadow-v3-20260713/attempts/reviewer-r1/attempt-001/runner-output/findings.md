## Findings

### FIND-001

- Severity: `error`
- Category: `test-design`
- ATOM: `ATOM-018`
- Test cases: `TC-ACPD-010`
- Проблема: Тестовые данные «значения заявки, необходимые для её сохранения» являются generic fixture. Кейс не задаёт исполнимую подготовку к сохранению и не позволяет проверить, что отображённое значение сопоставлено с ожидаемым ID клиента.
- Требуемое изменение: Задать конкретный доступный набор данных и проверяемое ожидаемое значение либо явно ограничить кейс наблюдаемым эффектом, который можно проверить без неподтверждённой технической атрибуции.

### FIND-002

- Severity: `error`

### FIND-003

- Severity: `error`
- Category: `expected-result`
- ATOM: `ATOM-021`
- Test cases: `TC-ACPD-013`
- Проблема: «Дата в допустимом диапазоне» и результат «предназначено для ввода даты» не являются конкретными данными и наблюдаемым oracle типа поля.
