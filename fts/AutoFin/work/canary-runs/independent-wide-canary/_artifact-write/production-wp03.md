## TC-AF43-AW-022
**Название:** Поле «Мобильный телефон» клиента отображается в карточке заявки
**Приоритет:** High
**Тип:** Positive
**package_id:** `WP-03`
**Трассировка:** `BSR 162`; Table 7 row 59; `ATOM-AW-048`

**Предусловия:**
1. В списке заявок нажать кнопку «Создать заявку».
2. В карточке заявки перейти к блоку «Контакты клиента».

**Тестовые данные:**
- Не требуются.

**Шаги:**
1. Найти поле «Мобильный телефон».

**Итоговый ожидаемый результат:** Поле «Мобильный телефон» отображается.

**Постусловия:**
- Не требуются.

## TC-AF43-AW-023
**Название:** Поле «Мобильный телефон» клиента отображает 10 цифр по шаблону с кодом страны
**Приоритет:** High
**Тип:** Positive
**package_id:** `WP-03`
**Трассировка:** `BSR 163`; `BSR 164`; Table 7 row 59; `ATOM-AW-049`; `ATOM-AW-050`

**Предусловия:**
1. В списке заявок нажать кнопку «Создать заявку».
2. В карточке заявки перейти к блоку «Контакты клиента».
3. Очистить поле «Мобильный телефон».

**Тестовые данные:**
- Номер без кода страны: `9234567890`.

**Шаги:**
1. Ввести в поле «Мобильный телефон» значение `9234567890`.

**Итоговый ожидаемый результат:** В поле «Мобильный телефон» отображается номер по шаблону с кодом страны `+7 (9xx) xxx-xx-xx`.

**Постусловия:**
- Очистить введенный мобильный телефон без сохранения заявки.
